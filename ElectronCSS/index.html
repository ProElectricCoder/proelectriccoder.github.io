ElectronCSS - A theme library

ElectronCSS is a theme library made by me, it currently contains different types of gradients which are not present in noremal CSS, such as Quadratic and cubic gradients. The background can be set like this: 

			
				import { cubicGradient } from 'https://proelectriccoder.github.io/ElectrocCSS/CubicGradient.js';
				document.body.style.background = cubicGradient({
					direction: 'to left',  // direction, all CSS linear-gradient direction are supported
					start: '#000055',      // start colour
					end: '#000000',        // end colour
					steps: 20,             // the more, the smoother
					power: 3               // 3 for cubic, 2 for quadratic, etc
				});
ElectronThemes.css currently contains the class theme-electron that gives the styles that are visible on this very page.
