Built to replace Chrome's blank new tab with something that actually works for you.

🎨
Cubic Gradient Background
Powered by ElectronCSS's cubicGradient — tune direction, power curve, and step count for a perfectly smooth gradient every time.
🔍
Multi-Engine Search
Switch between Google and Bing with one click. Smart URL detection navigates directly when you type a domain.
⚡
Quick Shortcuts
Add, edit, delete, and reorder shortcuts with drag-and-drop. Icons are fetched automatically from each site's favicon.
🕐
Live Clock & Date
12-hour monospace clock with AM/PM indicator and a full date line. Always centred, always sharp.
⚙️
Settings Sidebar
Slide-out panel gives you full control over gradient colours, direction, curve intensity, and smoothness — all live-previewed.
🔒
Fully Local
Zero network requests beyond search and favicons. All settings stored locally. No accounts, no tracking.

Get it

Download

Choose how you'd like to receive the extension files.

📂
Select a Folder
Uses the File System Access API to write all extension files directly into a folder you choose. No extraction needed — just point Chrome straight at it. Also links the folder for auto-updates.

🗜️
Download as ZIP
Downloads a NewTab-extension.zip containing all files. Unzip it somewhere permanent, then load that folder in Chrome.

Set it up

Installation

Four steps and you're done. Takes under a minute.

01
Download the extension files
Use the "Choose Folder" option above (recommended) or download and unzip the ZIP file into a permanent location. Don't move the folder after loading it.
02
Open Chrome Extensions
Navigate to the extensions management page.
chrome://extensions
03
Enable Developer Mode
In the top-right corner of the extensions page, toggle on Developer mode.
04
Load the extension
Click Load unpacked and select the folder. Open a new tab — you're done.
