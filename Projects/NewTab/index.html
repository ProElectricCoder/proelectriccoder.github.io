Set it up

Installation

Four steps and you're done. Takes under a minute.

01
Download the extension files
Use the "Choose Folder" option above (recommended) or download and unzip the ZIP file into a permanent location on your machine. Don't move the folder after loading it.
02
Open Chrome Extensions
Navigate to the extensions management page in your browser.
chrome://extensions
03
Enable Developer Mode
In the top-right corner of the extensions page, toggle on Developer mode. Three new buttons will appear in the top-left.
04
Load the extension
Click Load unpacked and select the folder containing the extension files. Open a new tab — you're done.
